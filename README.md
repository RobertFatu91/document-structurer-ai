# Document Structurer AI

Turn messy notes into structured reports instantly.

Document Structurer AI converts raw notes into organized reports automatically.

Paste messy notes and the AI extracts:

- Summary
- Key points
- Action items
- Structured sections

Perfect for meetings, brainstorming and messy planning notes.

## Try the tool

https://document-structurer-ai.vercel.app

## Features

- AI powered note structuring
- Clean structured reports
- Export as TXT
- Export as Word
- Export as PDF
- Fast and simple interface

## Built with

- Next.js
- OpenAI API
- Stripe
